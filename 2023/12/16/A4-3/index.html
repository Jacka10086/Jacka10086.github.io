Collections in Java (Java中的集合)

  • Java集合概述

    • Java提供了丰富的内置集合类型，通过Java类库并通过import语句包含。

  • 集合ADT的应用

    • 在大多数情况下，如果需要的集合ADT与内置类型不完全匹配，可以使用内置类型作为基础。

  • 主要集合类型

    • 特别是，在java.util库中，有两种集合类型可以被视为基础数组和链结构的扩展，用于实现Bag接口：
      • ArrayList 类
      • LinkedList 类

  • 总结

    • 介绍Java集合部分强调了作为程序员理解如何定义集合ADT以响应特定应用需求的重要性，以及如何利用Java内置的集合类型来实现这些数据结构。

本Markdown代码块总结了Java编程实践课程中关于Java集合的介绍，强调了集合类型的选择和实现在软件开发中的应用重要性。

类ArrayList（Class ArrayList）

  • 提供一个可调整大小的数组，克服了标准Array类型的许多限制。
    • ArrayList的元素是有序的，但可以通过add()方法添加元素。
    • 注意，ArrayList的类型是使用泛型符号指定的。

导入ArrayList类

  • 在程序的开始，我们需要导入ArrayList类。
    • import java.util.ArrayList;
    • 这行代码位于程序的最上方，告诉Java编译器我们将使用ArrayList类。

创建Demo类

  • 创建一个名为Demo的公共类。
    • public class Demo {}
    • 类是Java中的一个基本概念，用于定义对象的蓝图。

主函数入口

  • 在Demo类中定义主函数。
    • public static void main(String[] args) {}
    • 这是每个Java程序的入口点，程序从这里开始执行。

初始化ArrayList

  • 创建一个ArrayList实例用于存储字符串。
    • ArrayList<String> people = new ArrayList<String>();
    • ArrayList是泛型的，这里指定了它将存储字符串类型的元素。

添加元素

  • 使用add()方法向people列表中添加元素。
    • people.add("Adrian");
    • people.add("Belle");
    • people.add("Charles");
    • people.add("Delia");
    • add()方法将元素添加到列表的末尾。

打印ArrayList

  • 使用System.out.println()打印people列表。
    • System.out.println(people);
    • 输出将是列表中的所有元素。

总结：以上内容介绍了如何在Java中使用ArrayList类来存储有序的元素集合，并展示了如何初始化一个ArrayList实例、向其中添加元素以及打印出所有元素。ArrayList是一个非常灵活和常用的集合类，比标准数组更加强大，因为它可以动态调整大小并提供了更多的方法来操作集合数据。

类ArrayList（Class ArrayList）

  • 可以通过get()和set()方法访问元素，这两个方法都需要索引号作为参数。

访问元素

  • 通过get()方法可以获取指定位置的元素。
    • System.out.println(people.get(0));
    • System.out.println(people.get(2));
    • 这些方法调用返回列表中相应索引位置的元素。

修改元素

  • 通过set()方法可以设置指定位置的元素。
    • people.set(1, "Bonnie");
    • 这行代码将索引1处的元素从”Belle”更改为”Bonnie”。

打印ArrayList

  • 使用System.out.println()打印people列表。
    • 在修改元素后打印，可以看到列表元素已更新。
    • 输出显示列表中元素的当前状态。

总结：本部分内容进一步介绍了如何使用ArrayList的get()和set()方法来访问和修改列表中的元素。这显示了ArrayList的灵活性，能够轻松地在任何位置检索和更新元素。get()方法允许我们读取特定索引的元素，而set()方法则允许我们在特定索引处替换元素。

类ArrayList（Class ArrayList）

  • 元素可以通过remove()方法移除，该方法接受一个索引号作为参数。
  • remove()方法自动关闭数组中的空隙。

移除元素

  • 使用remove()方法可以删除指定位置的元素。
    • people.remove(2);
    • 这行代码将删除列表中索引为2的元素。

打印ArrayList

  • 在移除元素之前和之后使用System.out.println()打印people列表。
    • System.out.println(people.get(2));
    • 打印操作显示移除操作前后列表中元素的变化。

自动调整大小

  • ArrayList在元素被移除后会自动调整大小。
    • 移除元素后，后续元素会向前移动填补空位。

总结：本部分内容展示了如何使用ArrayList的remove()方法来删除列表中的元素，以及ArrayList如何在删除元素后自动调整大小和元素位置。这进一步证明了ArrayList相较于普通数组更加灵活的优势，特别是在需要动态添加和移除元素的场合。

类ArrayList（Class ArrayList）

插入元素

ArrayList 与 LinkedList 比较（ArrayList vs LinkedList）

ArrayList 类（ArrayList 类）

  • ArrayList 包含一个常规数组。当添加新元素且数组不够大时，会创建一个新的更大的数组，将原数组的元素复制过去，并移除旧数组。
  • ArrayList 适合存储和检索性质相对静态的数据。

LinkedList 类（LinkedList 类）

  • LinkedList 以链表的形式组织，每个元素都存储在一个容器中，每个容器都有一个指向列表中下一个容器的链接。
  • LinkedList 适合操作经常变化的数据。

总结：

这一页内容对 ArrayList 和 LinkedList 两种集合类型进行了比较。ArrayList 是基于数组结构，适合存取操作较为频繁的场景，但在列表中间添加或删除元素时效率较低。而 LinkedList 是基于链表结构，适合元素添加和删除操作较为频繁的场景，尤其是在列表的首尾。了解这两种数据结构的特点可以帮助我们根据需要选择合适的类型来存储和操作数据。
