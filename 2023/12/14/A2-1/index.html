COM410 编程实践 (COM410 Programming in Practice)

A2.1 变量和数据类型 (Variables and Data Types)

  • 在课程早期接触的语言中，不一定要求给变量分配特定类型。
  • Java是一种强类型语言：
    • 变量在使用前必须声明。
    • 变量声明时必须指定其能够保存的值的范围。
    • 一旦声明，变量的类型就不能改变。

Java数据类型 (Java Data Types)

  • 强类型语言的特点：
    • 必须在变量使用前声明。
    • 不能在未指定值域的情况下声明变量。
    • 变量类型一旦声明后不可更改。
  • Java变量声明示例：
    • int age = 25;
    • double pi = 3.14;
    • double accuracy = 99.5, percentage = 0.5;
    • boolean javaIsFun = true, isJavaFun;
  • 可以在单个语句中声明同一类型的多个变量。
  • 初始值是可选的。
  • 变量名遵循驼峰命名法（camelCase）。

总结：本节介绍了在Java编程实践中关于变量和数据类型的基本概念，强调了Java作为一种强类型语言的特性，包括变量声明的规则，变量类型的不可变性，以及如何在Java中声明和初始化变量。示例代码进一步阐释了如何在Java中声明不同类型的变量，并说明了命名约定。

Java数据类型 (Java Data Types)

Java提供了8种基本的变量类型。

基本类型 (Primitive types)

  • byte：8位整数
  • short：16位整数
  • int：32位整数
  • long：64位整数
  • float：32位浮点数
  • double：64位浮点数
  • boolean：布尔值（true或false）
  • char：单个字符值

示例 (Examples)

  • byte myVariable1 = 100;
  • short myVariable2 = 2000;
  • int myVariable3 = 32766;
  • long myVariable4 = 65536;
  • float myVariable5 = 3.14f;
  • double myVariable6 = 1234.567;
  • boolean myVariable7 = true;
  • char myVariable8 = 'c';

总结：这一节详细介绍了Java中基本数据类型的定义和每种类型的位大小。通过具体的代码示例，展示了如何在Java中声明和初始化各种基本类型的变量。这些基本类型是构建Java程序的基础，理解它们对于能够正确处理不同种类的数据至关重要。

字符串操作 (String Manipulations)

  • 返回字符串长度
  • 大小写转换
  • 查找和提取子字符串

示例 (Examples)

总结：本节阐释了Java字符串并非基本类型，而是一种对象，它提供了多种操作字符串的方法，包括获取字符串长度、大小写转换以及子字符串的查找和提取。示例代码演示了如何使用这些字符串操作方法，体现了Java中字符串处理的灵活性和功能性。

Java字符串操作（Java String Operations）

  • 字符串长度计算（Calculating String Length）

    • 使用length()方法获取字符串长度。
    • 示例：String myModule = "COM410 Programming in Practice";
    • 输出：Length is 30 characters

  • 转换为大写（Converting to Uppercase）

    • 使用toUpperCase()方法将字符串转换为大写。
    • 输出：As uppercase: COM410 PROGRAMMING IN PRACTICE

  • 转换为小写（Converting to Lowercase）

    • 使用toLowerCase()方法将字符串转换为小写。
    • 输出：As lowercase: com410 programming in practice

  • 查找字符位置（Finding Character Position）

    • 使用indexOf()方法查找字符或字符串首次出现的位置。
    • 查找”n”首次出现的位置。
    • 输出："n" first appears at position 15

  • 查找子字符串位置（Finding Substring Position）

    • 使用indexOf()方法查找特定子字符串下一次出现的位置。
    • 查找从位置16开始的”n”的位置。
    • 输出："in" next appears at position 19

  • 获取子字符串（Getting Substring）

    • 使用substring()方法获取指定位置的子字符串。
    • 获取从位置7到18的子字符串。
    • 输出：Substring from position 7 to 18 is Programming

总结

Java字符串操作是基础中的基础，它包括了各种操作字符串的方法，如计算长度、改变大小写、搜索字符以及提取子字符串。这些操作在日常编程中非常实用，是每个Java程序员必须掌握的技能。

Java字符串连接（Java String Concatenation）

  • 使用+操作符连接（Using the + Operator to Concatenate）

    • 在Java中，可以使用+操作符来连接字符串。
    • 需要注意，当连接字符串和数字时，结果可能会出乎意料。

  • 示例代码（Example Code）

    • 连接两个字符串变量。
    • String myModuleCode = "COM410", myModuleName = "Programming in Practice";
    • 输出：COM410 Programming in Practice

  • 字符串与数字的连接（Concatenating Strings and Numbers）

    • 连接字符串表示的数字。
    • String number1 = "10", number2 = "20";
    • 输出：1020（注意这里是字符串连接，不是数字加法）

  • 字符串与整数的连接（Concatenating Strings and Integers）

    • 连接字符串和整数类型的数字。
    • int number3 = 30;
    • 输出：1030（字符串”10”与整数30连接，而不是相加）

总结

在Java中使用+操作符连接字符串时，如果操作数中有字符串类型，那么+操作符将执行字符串连接操作，而不是数学上的加法。这意味着即使操作数看起来像数字，只要它们中的任何一个是字符串类型，连接的结果也会是字符串，而不是数值类型的和。因此，在进行操作时必须小心，以避免错误的结果。

Java数组基础（Java Arrays Basics）

  • 数组的定义和存储（Definition and Storage）

    • Java数组可以存储同一类型的多个值在一个命名变量中。
    • 通过[]符号定义和操作数组。

  • 数组的声明（Array Declaration）

    • 声明一个整型数组并直接初始化。
    • 示例：int[] myNumbers = { 10, 20, 30, 40, 50 };
    • 声明一个整型数组并预留空间，稍后初始化。
    • 示例：int[] yourNumbers = new int[10];
    • 打印数组中的元素。
    • 示例：System.out.println(myNumbers[2]); // 输出30

  • 修改数组元素（Modifying Array Elements）

    • 修改数组yourNumbers的第三个元素为100。
    • 示例：yourNumbers[2] = 100;
    • 打印修改后的元素。
    • 示例：System.out.println(yourNumbers[2]); // 输出100

  • 数组的注意事项（Array Cautions）

    • 预留数组空间即使我们不知道值。
    • 访问未初始化的数组元素时要小心。

  • 字符串数组的使用（Using String Arrays）

    • 声明并初始化一个字符串数组。
    • 示例：String[] myStrings = new String[3];
    • 打印数组中的元素，未赋值的元素默认为null。
    • 示例：System.out.println(myStrings[1]); // 输出null

总结

Java数组是一种存储固定大小的同类型元素的数据结构。我们可以在声明时直接初始化数组，或者先声明数组大小后再逐个赋值。在使用数组时，我们需要注意数组的索引是从0开始的，同时也要注意未赋值的数组元素会有默认值，例如整型数组的默认值是0，对象数组（如字符串数组）的默认值是null。数组是Java编程中处理一组数据时的基础工具。

Java多维数组（Java Multidimensional Arrays）

  • 多维数组的用途（Purpose of Multidimensional Arrays）

    • 多维数组对于表示表格形式的数据非常有用。
    • 数组的每个元素本身就是一个数组。
    • 理论上可以有高达255维的数组，但这会很难可视化和管理。

  • 示例代码（Example Code）

    • 声明并初始化一个二维整型数组。
    • 示例：int[][] myNumbers = { { 10, 20 }, { 30, 40 }, { 50, 60 } };
    • 打印二维数组中的元素。
    • 示例：System.out.println(myNumbers[1][0]); // 输出30

  • 声明并修改多维数组（Declaration and Modification of Multidimensional Arrays）

    • 声明一个二维整型数组并预留空间。
    • 示例：int[][] yourNumbers = new int[10][4];
    • 修改数组中的元素。
    • 示例：yourNumbers[7][2] = 100;
    • 打印修改后的元素。
    • 示例：System.out.println(yourNumbers[7][2]); // 输出100

注意事项（Cautions）

  • 多维数组可以通过连续的[]符号来访问和修改。
  • 在使用多维数组时，需要注意数组的维数和每个维度的长度，以避免数组越界等错误。

总结

多维数组提供了一种在Java中表示复杂数据结构的方法，特别是当我们需要表示多维数据时（如表格数据）。通过合理的使用和管理，多维数组可以使数据的存储和处理更加高效和结构化。但同时，编程时需要谨慎处理，确保不会访问到未初始化的数组元素或超出数组的边界。

Java随机数生成（Java Random Values）

  • 使用Random类（Using the Random Class）

    • 通过java.util库中的Random对象来生成随机数。
    • 必须先导入Random类。

  • 代码实现（Code Implementation）

    • 创建Random对象实例。
    • 示例：Random random = new Random();
    • 使用nextInt()方法生成随机整数。
    • 示例：int randomDigit1 = random.nextInt(bound: 10);

  • 多种数据类型的随机数（Random Values for Various Data Types）

    • nextInt()方法用于生成一个随机整数，小于指定的参数值。
    • 除了整数，Random类还提供生成其他数据类型随机数的方法，如long, double, byte, float, boolean等。

  • 输出示例（Output Example）

    • 打印出随机生成的三个数字。
    • 示例：System.out.printf("My randomly selected digits are %d %d %d\n", randomDigit1, randomDigit2, randomDigit3);
    • 输出：My randomly selected digits are 2 7 4

注意事项（Cautions）

  • 使用随机数生成器时，要确保导入了正确的类并且理解如何设置生成随机数的范围。
  • 同样要注意，随机数生成器是伪随机的，是基于一个确定的算法计算出来的，并不是真正意义上的“随机”。

总结

Random类是Java中生成随机数的一个实用工具，它可以用于生成各种数据类型的随机数。通过正确使用Random类的不同方法，我们可以在程序中实现各种基于随机数的功能，比如测试、模拟和游戏开发等。

场景（Scenario）

  • 创建一个名为TownChallenge的新Java项目，包含一个名为TownChallenge.java的类文件，在main()方法中实现以下功能：
    • 定义一个String数组，命名为towns，用来存储8个自选城镇的名称。
    • 生成并显示一系列体育赛事结果，使得towns[0]与towns[1]对战，towns[2]与towns[3]对战，以此类推。每个城镇赢得的分数应该是一个0-9范围内的随机整数。例如，结果显示可能是这样的：
      • Banbridge 7 Ballymena 4
      • Belfast 3 Newry 3
      • Coleraine 9 Enniskillen 0
      • Lurgan 5 Portadown 5

总结

本场景要求开发者创建一个Java项目，其中包含一个数组来存储城镇名称，并模拟这些城镇之间的比赛结果。结果是随机生成的，体现在每个城镇获取的分数上。

扩展说明

  • 这个Java项目的核心是处理数组和随机数生成。
  • 为每场比赛随机生成分数的方式可以利用Java的Random类或者Math.random()方法。
  • 显示的结果应该是格式化的字符串，每行显示一场比赛的结果和对应的分数。
  • 这个练习可能是为了教学目的，它提供了数组操作和随机数生成的实践机会。
