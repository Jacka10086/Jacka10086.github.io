总结

本页是奥斯特大学（Ulster University）的COM410编程实践课程的一部分，介绍了A2.2单元，即条件结构和循环的内容。

条件结构（Conditional Structures）

  • 使得代码执行的决策过程，或在多个代码片段之间进行选择
    • 提供了3种可供选择的条件结构：
      • if语句：当测试条件为真时，执行一个语句块。
      • if...else语句：当测试条件为真时，执行第一个语句块，否则执行第二个语句块。
      • switch语句：根据测试条件的不同值，执行多个语句块之一。

图表解释

  • 第一个图表代表if语句，当条件测试为真（True）时，执行一个语句块，假（False）时不执行。
  • 第二个图表代表if...else语句，根据条件测试结果为真还是假，选择执行两个语句块中的一个。
  • 第三个图表代表switch语句，它可以根据测试的值选择执行多个不同的语句块。

总结

条件结构是编程中用于控制代码执行流程的一种基础工具。通过条件测试的结果，选择执行特定的代码片段。if和if...else用于简单的二选一决策，而switch适用于多个可能结果的情况。

扩展说明

  • 条件结构是编程中实现逻辑决策和分支的基本构件。
  • 在编写条件结构时，重要的是清晰地定义条件测试，确保它们正确反映了需要做出决策的逻辑。
  • switch语句通常用于替代多个if...else语句，使代码更清晰和易于管理。

if语句（if Statement）

  • 条件执行块：一个语句块的执行取决于逻辑条件的结果。

    • 这意味着只有当条件为真时，if语句内的代码块才会被执行。

  • 导入Scanner类：程序首先导入了Scanner类，以便从用户处接收输入。

    • import java.util.Scanner;

  • 主方法：main方法是Java程序的入口点。

    • public static void main(String[] args) {
  Scanner keyboard = new Scanner(System.in);
  System.out.print("Please enter a number > ");
  int value = keyboard.nextInt();

  if (value % 2 == 0) {
    System.out.printf("%d is an even number.\n", value);
  }
}
    • 程序请求用户输入一个数字，然后检查它是偶数还是奇数，并输出相应的信息。

  • if…else结构：使用if...else语句来根据条件判断并输出结果。

    • 如果value % 2等于0，表示它是偶数；否则，它是奇数。

  • 示例输出：

    • 用户输入54，输出：”54 is an even number.”
    • 用户输入17，输出：”17 is an odd number.”

if...else语句在Java中用于在两种情况下进行选择。本例中，程序通过检查用户输入的数字是否能被2整除来决定输出是“偶数”还是“奇数”。这种结构允许程序能够根据条件的真假，执行不同的代码块。

循环结构（Loop Structures）

循环结构简介

  • 循环结构 允许代码重复执行，直到达到一定的限制或特定条件为真。

三种循环结构

  • for 循环
    • 通常用于当你知道循环应该执行的次数时。
  • while 循环
    • 如果不确定循环应该执行多少次，但有一个继续条件时使用。
  • do…while 循环
    • 类似于 while 循环，但至少执行一次，即使条件第一次测试就为假。

循环结构的流程图

  • For 循环流程图
    • 显示从语句块开始，经过循环再回到语句块，形成一个闭环。
  • While 循环流程图
    • 从测试条件开始，如果条件为真，则执行语句块然后再次测试条件；如果条件为假，则退出循环。
  • Do…While 循环流程图
    • 从语句块开始执行，执行完毕后测试条件，如果条件为真，则循环继续；如果为假，则停止。

总结

循环结构是编程中非常重要的部分，它允许代码在满足某些条件的情况下重复执行。有三种基本的循环结构：for循环、while循环和do…while循环，每种循环结构有其特定的使用场景和流程。流程图可以帮助我们理解每种循环的工作原理和逻辑流程。
