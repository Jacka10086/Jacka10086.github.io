B3.3 散列 (Hashing)

散列定义 (Definition of Hashing)

  • 散列是一种数据检索方法，它通过使用特定的算法（即散列函数）来将输入（如字符串）转换成固定大小的字符串（即散列值）。
  • 通过散列函数处理数据可以快速定位数据项，无需遍历整个数据集。

散列的优势 (Advantages of Hashing)

  • 散列能够提供快速的数据检索速度。
  • 每个数据项都有一个独一无二的地址，这个地址由散列值指定，从而使得数据的存取效率非常高。

散列的应用 (Applications of Hashing)

  • 散列广泛应用于数据库索引创建、数据压缩、加密算法等领域。
  • 在编程中，散列表（哈希表）是实现快速数据访问的重要数据结构。

散列的原理 (Principle of Hashing)

  • 散列过程涉及将大范围的键转换成小范围的索引。
  • 散列值（hash）是通过散列函数生成的，它通常是一个较短的随机字符串，用于在散列表中定位元素。

散列的挑战 (Challenges of Hashing)

  • 碰撞（两个不同的输入产生相同的散列值）是散列中的一个主要挑战。
  • 解决碰撞的方法包括链地址法、开放地址法等。

总结 (Summary)

  • 这一部分介绍了散列的基本概念、优势、应用、原理以及面临的挑战。
  • 散列是一种高效的数据检索技术，它通过特定的散列函数将数据映射到一个固定大小的散列值。
  • 散列在数据结构中有着广泛的应用，尤其是在快速检索和数据管理方面。

散列 (Hashing)

Java的HashMap (Java’s HashMap)

  • Java提供了一种名为HashMap的结构，它允许我们将一个项目的地址指定为一个变量（这个变量必须是Object类型）。
    • 例如，模块通过它们的模块代码（例如COM410）来定位，学生通过他们的B-Number定位，手机通过它们的唯一编号来定位等。

散列的操作 (Operation of Hashing)

  • 在散列中，不需要迭代或搜索来定位一个项目，项目要么存在于指定的地址，要么不存在。

总结 (Summary)

  • 这一节介绍了Java中HashMap的用途和散列的基本操作。在Java的散列实现中，可以直接通过键（Key）来快速访问值（Value），无需遍历整个集合，这大大提高了数据检索的效率。

Java HashMap (Java散列表)

HashMap的声明 (Declaring a HashMap)

  • 在Java中声明一个HashMap需要指定键（Key）和值（Value）的对象类型。
    • 使用import java.util.HashMap;来导入HashMap类。
    • 声明示例：HashMap<String, Integer> populations = new HashMap<String, Integer>();

键值对类型 (Types of Key-Value Pairs)

  • 在这个例子中，键是一个字符串（String），值是一个整数（Integer）。

总结 (Summary)

  • 本节展示了如何在Java中声明一个HashMap以及如何指定键值对的数据类型。HashMap是Java中一个非常有用的集合对象，能够以键值对的形式存储和管理数据，使得数据检索变得快速而简便。

Java HashMap（Java 哈希图）

put() 方法（添加方法）

  • put() 方法用于向 HashMap 中添加元素。
    • 参数类型必须与 HashMap 定义中的参数类型相匹配。

示例代码（代码示例）

  • populations.put("Dublin", 1024027); 将 “Dublin” 和 1024027 作为键值对添加到 populations 哈希图中。
  • populations.put("Belfast", 345418); 添加 “Belfast” 和 345418。
  • populations.put("Cork", 198304); 添加 “Cork” 和 198304。
  • populations.put("Dun Laoghaire", 185400); 添加 “Dun Laoghaire” 和 185400。
  • populations.put("Limerick", 90054); 添加 “Limerick” 和 90054。

输出结果（输出结果）

  • System.out.println("HashMap: " + populations); 打印出 populations 哈希图的内容。
    • 输出内容为 HashMap 的键值对，格式为 {key=value}。

总结：本节介绍了Java中 HashMap 类的 put() 方法，用于添加键值对到 HashMap 集合中。示例代码展示了如何添加城市名称和对应的人口数量到哈希图，以及如何打印出哈希图中的所有内容。

Java HashMap（Java 哈希图）

get() 方法（获取方法）

  • get() 方法用于通过键值检索 HashMap 中的元素。

示例代码（代码示例）

  • Integer pop = populations.get("Belfast"); 通过键 “Belfast” 从 populations 哈希图中获取人口数。
  • System.out.println("\nPopulation of Belfast is " + pop); 打印出 “Belfast” 的人口数。

输出结果（输出结果）

  • 输出结果显示为 “Population of Belfast is 345418”，即 “Belfast” 的人口数为 345418。

总结：本节讲述了如何使用 HashMap 中的 get() 方法通过键来检索对应的值。提供的代码示例演示了如何获取并打印出哈希图中的特定城市的人口数量。

Java HashMap（Java 哈希图）

检索数据（数据检索）

  • keySet() 方法获取所有键的集合。
  • values() 方法获取所有值的集合。
  • entrySet() 方法获取键值对的集合。

示例代码（代码示例）

  • System.out.println("\nPopulations are available for " + populations.keySet()); 打印所有可用的键。
  • System.out.println("The population values are: " + populations.values()); 打印所有的值。
  • System.out.println("The mappings are: " + populations.entrySet()); 打印所有的键值对。

输出结果（输出结果）

  • Populations are available for [Dun Laoghaire, Belfast, Cork, Dublin, Limerick] 显示所有的键。
  • The population values are: [185400, 345418, 198304, 1024027, 90054] 显示所有的值。
  • The mappings are: [Dun Laoghaire=185400, Belfast=345418, Cork=198304, Dublin=1024027, Limerick=90054] 显示所有键值对。

总结：本节介绍了 HashMap 中 keySet()、values() 和 entrySet() 三个方法，这些方法分别用于检索键集、值集和键值对集。示例代码说明了如何使用这些方法来打印出 HashMap 中存储的数据，并且输出结果部分清晰地展示了这些数据的实际内容。

Java HashMap（Java 哈希图）

键对应单一值（键值唯一性）

  • 在 HashMap 中，每个键只能对应一个值。
    • 如果对同一个键使用 put() 方法两次，后一次的值会替换前一次的值。

示例代码（代码示例）

  • populations.put("Belfast", 400000); 使用新的人口数 400000 更新 “Belfast” 的值。
  • System.out.println("\nPopulation of Belfast is " + populations.get("Belfast")); 打印更新后的 “Belfast” 人口数。
  • System.out.println("The mappings are: " + populations.entrySet()); 打印所有键值对，显示更新后的 “Belfast” 人口数。

输出结果（输出结果）

  • 输出显示 “Population of Belfast is 400000”，表明 “Belfast” 的值已更新为 400000。
  • 显示更新后的哈希图内容：“The mappings are: [Dun Laoghaire=185400, Belfast=400000, Cork=198304, Dublin=1024027, Limerick=90054]”。

总结：本节说明了 HashMap 中的键值唯一性原则，并展示了如何通过 put() 方法更新现有键的值。示例代码演示了更新 “Belfast” 的人口数，并通过打印语句验证了更新效果。

Java HashMap（Java 哈希图）

删除值（值的删除）

  • remove() 方法用于删除 HashMap 中的键值对。

示例代码（代码示例）

  • Integer belfastPop = populations.remove("Belfast"); 删除键为 “Belfast” 的键值对，并返回被删除值。
  • System.out.println("\nPopulation of Belfast is " + belfastPop); 打印被删除的 “Belfast” 人口数。
  • System.out.println("The mappings are: " + populations.entrySet()); 打印删除操作后的所有键值对。

输出结果（输出结果）

  • 输出结果显示 “Population of Belfast is 400000”，表明 “Belfast” 的人口数在删除前是 400000。
  • The mappings are: [Dun Laoghaire=185400, Cork=198304, Dublin=1024027, Limerick=90054] 显示删除 “Belfast” 后的哈希图内容。

总结：本节讲解了如何使用 remove() 方法从 HashMap 中删除特定的键值对。示例代码演示了删除 “Belfast” 的过程，并通过打印语句显示了该操作的结果。

检索不存在的键值（键值不存在时的检索）

  • 当尝试检索一个不存在的键对应的值时，HashMap 会返回 null。

示例代码（代码示例）

  • System.out.println("\nPopulation of Belfast is " + populations.get("Belfast")); 尝试打印 “Belfast” 的人口数，假设 “Belfast” 已被删除或未曾添加。

输出结果（输出结果）

  • 输出结果为 “Population of Belfast is null”，表明 “Belfast” 键不存在，因此对应的值为 null。

总结：本节阐述了 HashMap 的一个重要特性：当使用 get() 方法检索一个不存在的键时，会得到 null 结果。示例代码尝试获取 “Belfast” 的人口数，但因为此键可能已被删除或未添加，所以返回 null。
