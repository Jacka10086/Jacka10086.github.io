私有变量与公共变量的区别

在 Java 编程中，私有（private）和公共（public）是两种主要的访问修饰符，它们定义了类成员（变量和方法）的访问权限。

私有变量 (Private Variables)

  • 定义：只能在其定义的类中访问的变量。
  • 目的：封装类的实现细节，防止外部代码意外或恶意地修改类的内部状态。
  • 示例：在类中定义 private String name;，则只能在该类内部访问 name 变量。

公共变量 (Public Variables)

  • 定义：可以被任何其他类访问的变量。
  • 目的：为其他类提供直接访问该变量的能力，不需要通过方法。
  • 示例：在类中定义 public int age;，则任何外部类都可以直接通过 object.age 访问和修改。

注意事项

虽然使用公共变量可以提供直接的访问和修改方式，但为了保护类的内部状态不被意外或恶意地修改，通常建议使用私有变量，并为其提供公共的 getter 和 setter 方法。这种做法被称为封装，是面向对象编程的核心概念之一。

Pet.java

解释：this.name = theName;`与 this.age = theAge;

这两个赋值语句通常出现在构造函数中，用于将传递给构造函数的参数值分配给类的成员变量。

背景

假设有一个 Pet 类，其中包含两个成员变量：name 和 age，并且它们被声明为私有变量。构造函数 Pet(String theName, int theAge) 被设计为接收两个参数：theName 和 theAge。

使用 this 关键字

this 是一个特殊的关键字，用于引用当前对象的成员。

  • this.name 和 this.age 分别引用 Pet 类中的 name 和 age 成员变量。
  • 通过使用 this，即使参数名称与成员变量名称相同，我们也可以清晰地区分它们。

代码示例

解读赋值

this.name = theName;：表示将构造函数的第一个参数 theName 的值分配给 Pet 类的成员变量 name。
this.age = theAge;：表示将构造函数的第二个参数 theAge 的值分配给 Pet 类的成员变量 age。

解释：public void setName(String theName) { this.name = theName; }

public void setName(String theName) 是一个方法，设计用于设置 Pet 类的 name 成员变量的值。

方法细节

该方法执行以下操作：

  1. 接受一个字符串参数 theName。
  2. 使用 this.name 来引用 Pet 类的成员变量 name。
  3. 将 theName 分配给 this.name，从而更新 Pet 对象的 name 成员变量的值。

Cat.java

解释：public Cat(String theName, int theAge, String theBreed) { super(theName, theAge); }

这是 Cat 类的构造函数，用于初始化 Cat 对象。

构造函数细节

  • public Cat(String theName, int theAge, String theBreed) 接受三个参数：

    1. theName: 表示猫的名字。
    2. theAge: 表示猫的年龄。
    3. theBreed: 表示猫的品种。

  • super(theName, theAge) 是调用父类 Pet 的构造函数的语句。这里的 super 关键字用于调用父类的构造函数。

    • 通过此语句，theName 和 theAge 参数值被传递给 Pet 类的构造函数，以初始化 name 和 age 成员变量。

背景信息

Cat 类是 Pet 类的子类，继承了 Pet 类的属性和方法。在面向对象编程中，当创建子类的对象时，父类的构造函数也会被调用，确保所有的继承属性都被正确初始化。

Dog.java

this.dogBreed = theBreed; 语句解释

赋值语句的作用

  • this.dogBreed = theBreed; 是 Dog 类构造函数中的赋值语句。
  • 该语句用于将传递给构造函数的参数值 theBreed 分配给类的成员变量 dogBreed。

成员变量与构造函数的关系

  • Dog 类有一个成员变量 dogBreed，它被声明为私有变量。
  • 构造函数 Dog(String theName, int theAge, String theBreed) 接受三个参数：theName、theAge 和 theBreed。
  • 这些参数值被分配给 name、age 和 dogBreed 成员变量。

使用 this 关键字的意义

  • this 关键字用于引用当前对象的成员变量。
  • 在这个例子中，this.dogBreed 引用了 Dog 类的成员变量 dogBreed。
  • 使用 this 可以确保即使参数名称与成员变量名称相同，也能清晰地区分它们。

赋值的实际意义

  • this.dogBreed = theBreed; 语句的目的是将构造函数的参数 theBreed 分配给 Dog 类的成员变量 dogBreed。
  • 这样做的主要目的是存储狗的品种信息，这在 speak() 方法中可能会被使用。

PetTest.java

对象创建语句解释

创建 Cat 和 Dog 对象的语句

  • Cat c1 = new Cat("Pixel", 8, "Tabby");
  • Dog d1 = new Dog("Rex", 10, "Terrier");

使用 new 关键字

  • 使用 new 关键字创建了两个对象 c1 和 d1。
  • 这两个对象分别代表一只名为 “Pixel”、年龄为 8 岁、品种为 “Tabby” 的猫，以及一只名为 “Rex”、年龄为 10 岁、品种为 “Terrier” 的狗。

调用构造函数

  • 在创建对象时，我们调用了 Cat 和 Dog 类的构造函数，并传递了相应的参数。
  • 例如：new Cat("Pixel", 8, "Tabby") 调用了 Cat 类的构造函数，并传递了三个参数 “Pixel”、8 和 “Tabby”。
  • 这些参数用于初始化 Cat 对象的成员变量 name、age 和 (可能的) catBreed。

对象的使用

  • 创建对象后，我们可以使用类中定义的方法和成员变量。
  • 例如，可以使用 c1.meow() 调用 Cat 类的 meow() 方法。
  • 同样，可以使用 d1.bark() 调用 Dog 类的 bark() 方法。
  • 这些方法返回相应的字符串，表示猫或狗发出的声音。
