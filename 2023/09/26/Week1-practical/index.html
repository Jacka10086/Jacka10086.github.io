初始化实例变量

解释

这段代码在某个类（例如 Building 类）的构造函数或方法中，为两个实例变量 address 和 owner 分别设置了初始值。

核心操作

  • this 关键字: 在Java中，用于引用当前对象的实例变量或方法。
  • “”: 表示将字符串变量的值设置为一个空字符串。

详细步骤

  1. 引用实例变量: 使用 this.address 和 this.owner 分别引用当前对象的 address 和 owner 实例变量。
  2. 设置初始值: 使用 "" 将这两个实例变量的值设置为空字符串。
