W3 (LEC) - SaaP - SaaS

Created2023-10-10|Updated2023-10-25|Com435
|Post Views:

Software as a Product (软件作为产品)

SaaP (软件作为产品)

  • Application / Tool to solve a problem (应用/工具来解决问题)

    • 这是专为特定问题或需求设计的应用或工具。

  • Usually one off purchase via a license (high price) (通常是通过许可证一次性购买（高价）)

    • 用户通常需要支付一次性费用来购买并使用这个软件。

  • Major updates = another purchase (主要更新 = 另外的购买)

    • 当软件有重大更新时，用户可能需要再次购买。

  • Traditional approach (传统方法)

    • 这是早期软件发布和销售的常见方式。

  • Download / Install (下载/安装)

    • 用户需要下载并安装软件到他们的设备上。

Software as a Service (软件作为服务)

SaaS (软件作为服务)

  • Web App / Tool to solve a problem (Web应用/工具来解决问题)

    • 这是一个在线的应用或工具，可以通过互联网来访问和使用。

  • Subscription model (lower price) - pay as you go (订阅模型（低价）- 随用随付)

    • 用户按周期（如每月或每年）支付费用来访问和使用服务。

  • Dominate approach - 86% companies running SaaS (主导方法 - 86%的公司运行SaaS)

    • 目前，大多数公司都采用SaaS模型，这已经成为一种主导的趋势。

  • Cloud (云)

    • SaaS应用通常在云中托管，用户可以在任何地方访问。

  • Microsoft

  • Examples of SaaS companies

SaaS vs SaaP (Traditional Software) (SaaS与SaaP（传统软件）的对比)

  • SaaS offers free trial (SaaS提供免费试用)

    • 多数SaaS产品在购买前提供一段免费试用期。

  • Recurring payment (重复支付)

    • 与SaaP的一次性支付不同，SaaS通常需要周期性的支付。

  • No install needed (无需安装)

    • 用户无需下载或安装任何软件，直接通过浏览器访问。

  • Automatic updates (自动更新)

    • 所有的更新都会自动应用，无需用户干预。

  • Multiple users (多用户)

    • 多个用户可以使用同一帐户或许可。

  • Manages security / data backups (管理安全性/数据备份)

    • SaaS提供商通常负责数据的安全和备份。

  • Distributed via cloud (通过云分发)

    • SaaS应用是在云上托管的，确保高可用性和可扩展性。

  • Laptop, tablet, mobile (笔记本电脑，平板电脑，移动设备)

    • 用户可以在多种设备上访问和使用SaaS应用。
