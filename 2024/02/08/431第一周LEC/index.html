PHP简介

  • PHP最初是”Personal Home Page”的缩写，现在代表”HyperText Preprocessor”。
  • PHP是一种主要用于开发服务器端Web应用程序的编程语言。
  • 它是一种开源语言，可用于HTML代码中，常与MySQL等技术结合使用，创建互动式网站。

通过这个介绍，学生将对PHP有一个基本的认识，知道它是一种能够创建动态和交互式网页的服务器端语言。

PHP的历史

  • PHP 1：1994年6月由Rasmus Lerdorf开发。
  • PHP 3：1997年，由Andi Gutmans和Zeev Suraski发布了新版本PHP 3，引入了许多新特性和改进，包括对面向对象编程的支持。
  • PHP 4：2000年发布，进一步增强性能和改善Web服务支持。
  • PHP 8：2020年发布，带来了JIT编译器、联合类型、匹配表达式等新功能。

这部分内容讲述了PHP的发展历程，让学生理解这门语言是如何随着时间进步和技术的发展而不断演化的。

PHP的应用

  • 内容管理系统（CMS）
  • 电子商务网站
  • 数据可视化工具
  • 应用程序接口（APIs）

这一部分向学生展示了PHP在构建各种类型的Web应用程序中的多样性和广泛应用，强调了学习PHP的实际价值和重要性。

在WEB开发中，PHP与HTML的结合使用是实现动态网页的关键。以下是使用PHP进行WEB开发时的一些基本概念：

处理表单和GET/POST数据

  • 创建HTML表单以收集用户输入。
  • 在PHP中接收并处理表单数据。
  • 了解并使用PHP中的超全局变量$_GET和$_POST来获取请求数据。

操作文件和数据库

  • 连接到数据库，如MySQL或SQLite，并使用PHP进行数据的查询、插入、更新和删除。

这些技能对于任何希望开发现代Web应用程序的PHP开发人员来说都是基础且必须掌握的。

HTTP请求处理流程 - PHP/MySQL

以下是使用PHP和MySQL在服务器端处理HTTP请求的步骤说明：

  1. 用户计算机:

    • 通过浏览器访问 http://localhost/login.php。

  2. 请求:

    • 浏览器将HTTP请求发送到互联网。

  3. 互联网:

    • 请求通过互联网路由到Web主机。

  4. 硬盘:

    • Web服务器从硬盘检索请求的PHP文件。

  5. Web服务器:

    • 收到请求后，调用PHP解释器处理PHP文件。

  6. PHP解释器:

    • 解释器执行PHP脚本。

  7. MySQL数据库:

    • 如果脚本需要数据库数据，它会查询MySQL数据库。

  8. 数据库响应:

    • MySQL处理查询并将结果发送回PHP。

  9. PHP处理:

    • PHP脚本执行完毕，生成HTML响应内容。

  10. 响应传输:

    • Web服务器将HTML内容作为HTTP响应通过互联网发送。

  11. 用户计算机接收响应:

    • 浏览器接收响应并显示页面内容。

这个流程图清晰地展示了从用户发起请求到服务器处理并返回响应的全过程，涵盖了Web开发中的关键后端技术。

PHP变量

变量在PHP中用于存储值，并且它们的特点包括：

  • 以$符号开始，后跟变量名。
  • 变量名必须以字母开头。
  • 变量必须在PHP标签内声明。
  • PHP变量是松散类型的，可以动态地赋予任何类型的值。
  • 可以将任意值赋给变量。
  • 可以将值赋给包含不同类型值的其他变量，例如将整数赋值给字符串变量将重写并转换变量类型。

变量的类型

PHP支持多种基本数据类型，包括：

  • 整数（Integer）：用于存储整数值，例如 -4, 3。
  • 浮点数（Float）：用于存储有小数点的数值，例如 3.14。
  • 字符串（String）：用于存储字符序列，例如 "COM431"。
  • 布尔值（Boolean）：用于存储 true 或 false。
  • 数组（Array）：用于存储多个数据项。
  • 对象（Object）：用于存储类的实例。

了解PHP中的变量类型对于编写有效的代码和进行数据操作至关重要。

整数 (Integers)

  • 基础数据类型 (Basic data type)
    • 整数是编程中最基础的数据类型之一，用于表示没有小数部分的数字。
  • 用于数学运算 (Used in mathematical operations)
    • 整数可以用在加减乘除等各种数学运算中。
  • 可以是正数、负数或零 (Can be positive, negative or zero)
    • 整数包括所有的正整数、负整数以及零。

PHP中的整数 (Integers in PHP)

  • PHP中的整数是没有小数点的全数字 (Whole numbers without decimal points)
    • 在PHP中，整数是指那些既可以是正数也可以是负数，且不包含小数点的数字。
  • 使用”int”数据类型表示 (Represented using the “int” data type)
    • PHP通过”int”这个数据类型来存储整数值。

在数学运算中的应用 (Used in mathematical operations)

  • 用于加法、减法、乘法和除法 (Addition, subtraction, multiplication, and division)
    • 整数常用于编程中的基本数学计算，如加法（+）、减法（-）、乘法（*）和除法（/）。

在循环和条件语句中的应用 (Used in loops and conditional statements)

  • 可以在循环和条件语句中使用 (Can be used in loops and conditional statements)
    • 在PHP编程中，整数常被用作循环的计数器或者条件语句中的比较值。

例子 (Example)

  • $e = $a * $b; 计算a和b的乘积。
  • $f = $a / $b; 计算a除以b的商。
  • $g = $a % $b; 计算a除以b的余数（注：此操作通常用于整数，对于浮点数，它可能不会返回有效的结果）。

字符串 (Strings)

  • 基本数据类型 (Basic data type)
    • 字符串是用于表示一系列字符的基本数据类型。
  • 用于表示字符序列 (Used to represent a sequence of characters)
    • 字符串可以包含字母、数字、符号等字符组成的序列。
  • 可以用单引号(‘)或双引号(“)括起来 (Can be enclosed in single or double quotes)
    • PHP中的字符串可以用单引号或双引号表示，两者有细微的差别，比如双引号内可以解析变量。
  • 可以使用点(.)运算符连接 (Can be concatenated using the dot operator)
    • 在PHP中，可以使用点(.)运算符将两个字符串连接起来，形成一个新的字符串。

PHP中字符串的内置函数 (Different built-in functions for strings in PHP)

  • 比如strlen、str_replace和substr等 (e.g., strlen, str_replace, and substr)
    • strlen函数用来获取字符串的长度。
    • str_replace函数用来替换字符串中的某些部分。
    • substr函数用来提取字符串的子串。

代码示例 (Example code)

  • $name = "John Smith"; 初始化了一个名为$name的字符串变量，其值为”John Smith”。
  • $greeting = "Hello, "; 初始化了一个名为$greeting的字符串变量，其值为”Hello, “。

  • 使用echo $message;输出拼接后的字符串，结果为”Hello, John Smith”。

数组 (Arrays)

数据结构 (Data structure)

  • 数组是一种数据结构，用于存储一系列的值，这些值可以是不同类型的，例如整数、字符串或对象。
    • 这意味着数组可以包含多种类型的元素，从简单的数字和字符串到复杂的对象和其他数组。

定义数组 (Defining arrays)

  • 数组可以通过array()函数或使用方括号[]来定义。
    • 使用array()函数可以创建数组并初始化其值。
    • 方括号[]是另一种更简洁的语法，用于定义数组。

数组中的每个值 (Each value in an array)

  • 数组中的每个值都被赋予一个唯一的索引。
    • 索引是用于访问数组中特定元素的数字或字符串。

索引的使用 (Using indexes)

  • 索引可以用来访问该值。索引可以是一个整数或一个字符串。
    • 整数索引通常用于有序集合，如列表。
    • 字符串索引（或键）用于关联数组，其中每个值都有一个具体的名称。

总结 (Summary)

  • 在PHP中，数组是一种包容性强的数据结构，既可以通过使用array()函数定义，也可以简单地使用方括号[]来创建。数组的元素可以通过唯一的索引进行访问和操作，这些索引可以是整数或字符串。在编程中，理解和有效使用数组对于管理和操作集合数据至关重要。

数组示例 (Examples of Arrays)

整数数组 (Array of integers)

  • $numbers = array(1, 2, 3, 4, 5);
    • 这是一个基本的数组，包含了一系列的整数。

关联字符串数组 (Associate array of strings)

  • $colors = array("red" => "#f00", "green" => "#0f0", "blue" => "#00f");
    • 关联数组使用键值对存储数据，每个键对应一个特定的值。

索引字符串数组 (An indexed array of strings)

  • $fruits = array("apple", "banana", "cherry");
    • 这是一个索引数组，其中每个元素自动分配一个数字索引。

数组的关联数组 (Associative array of arrays)

  • $person = array("name" => "John", "age" => 30, "address" => array("street" => "123 Main St", "city" => "Anytown", "state" => "CA", "zip" => "12345"));
    • 这是一个更复杂的结构，一个关联数组内包含另一个数组，形成了多维数组。

总结 (Summary)

  • 数组是PHP中用于存储数据集合的结构。整数数组按顺序存储数字，关联数组则使用键值对来存储数据，允许你通过键名来访问值。索引数组自动为每个元素分配一个索引，而多维数组则是数组中嵌套数组的结构。这些数组的示例展示了不同类型的数组及其在PHP中的用法。

使用数组索引 (Using array index)

  • $fruits = array("apple", "banana", "cherry");
  • echo $fruits[0]; // 输出 "apple"
    • 通过指定索引来访问数组中的元素。

使用foreach循环 (Using foreach loop)

  • foreach ($fruits as $fruit) { echo $fruit . "<br>"; }
    • foreach循环遍历数组中的每个元素，并输出。

使用array_values()函数 (Using array_values())

  • $values = array_values($fruits);
  • echo $values[0]; // 输出 "apple"
    • array_values()函数返回数组中所有的值，这些值构成了一个新的数字索引数组。
