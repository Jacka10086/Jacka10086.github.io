PHP 条件语句 (PHP Conditional Statements)

在 PHP 编程中，条件语句用于基于不同的条件来执行不同的代码块。

if 语句 (If Statement)

  • if 语句是最简单的条件语句，它在条件为真时执行代码块。
  • 只有当测试条件为 True 时，才会执行 if 代码块内的语句。

if…else 语句 (If…Else Statement)

  • if...else 语句在 if 条件为 True 时执行代码块，否则执行 else 代码块。
  • 这提供了两种不同的执行路径。

if…elseif…else 语句 (If…Elseif…Else Statement)

  • if...elseif...else 语句可以根据多个条件执行多个不同的代码块。
  • 这是一种更复杂的条件语句，允许多个测试条件和相应的代码块。

switch 语句 (Switch Statement)

  • switch 语句根据不同的条件值执行不同的代码块。
  • 它是另一种条件语句，可以根据一个变量的不同值来执行不同的代码块。

在决策过程中，选择使用哪种条件语句取决于特定的需求和条件的复杂性。

总结：PHP 提供了多种条件语句，以便开发者可以根据不同的条件执行不同的代码路径。这是编程中控制流程的基础。

超文本传输协议（HyperText Transfer Protocol）

  • HTTP 协议 - 万维网的基础协议，定义了消息如何被格式化和传输，以及网络服务器和浏览器应该如何响应不同的命令。

    • 无状态协议

      • HTTP 是一个无状态协议，这意味着它不会保留有关用户以前请求的任何信息。
      • 每个客户端到服务器的请求都被视为新的，之前的请求不会被连续链接。

    • 数据类型协商

      • HTTP 允许进行数据类型和表示形式的协商，以便通过万维网连接的系统可以以所有系统都能理解的格式交换信息。

    • 应用层协议

      • HTTP 是为分布式、协作和超媒体信息系统设计的应用层协议。
      • 它不仅用于获取 HTML、图像和视频，还用于提交表单数据、上传文件和其他网络客户端（如浏览器）与服务器之间的交互。

  • 通信过程示例

    • 一个 HTTP 客户端（如 web 浏览器）发送 HTTP 请求到服务器，然后服务器以 HTTP 响应消息回应。
    • 通信通过 TCP/IP 协议完成，这是一套用于互联网上网络设备间的通信协议。
  • PHP 变量
    • name，age，和 campus 变量被赋值并嵌入在 HTML 中。
    • 使用 echo 语句在 HTML 中输出 PHP 变量的值。
