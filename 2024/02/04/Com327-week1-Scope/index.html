Python 作用域简介（Introduction to Python Scope）

在本节中，我们将介绍 Python 编程中“作用域”的概念。作用域指的是程序中可以访问变量的区域。理解作用域对于变量管理至关重要，尤其是在处理函数和区分局部变量与全局变量时。

  • 全局作用域（Global Scope）

    • 在模块的顶层或任何函数之外声明的变量处于全局作用域。

作用域解析顺序（Scope Resolution Order - LEGB Rule）

Python遵循LEGB规则进行名称解析：Local, Enclosing, Global, Built-in。这意味着Python首先在局部作用域中查找变量名，然后是封闭作用域，其次是全局作用域，最后是内建作用域。本部分演示了LEGB规则的实际操作。
